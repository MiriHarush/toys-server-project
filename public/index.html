Toys api documentation

User-Routes

#Method

POST

URL

/users

#Body

{
"name" : "Miri Harush",
"email" : "miri@gmail.com",
"password" : "1234",
"date_created" : "2023-11-30", /* not require */
"role" : "User" /* not require */
}

#Description

Create new user

#Method

POST

URL

/users/login

#Body

{
"email" : "miri@g.com",
"password" : "1234",
}

#Description

User login, if the email and password are correct the server will respond with a token, If not, an appropriate message will be sent to the user

Toys-Routes

#Method

GET

URL

/toys/all

#Body

none

#Description

Get a list of all the toys, will present 10 results per page.

#Method

GET

URL

/toys/search?s=Fun Science Experiment Kit

#Body

none

#Description

search for a toy by searching follwing name or info after the "s=", it will present 10 results per page.

#Method

GET

URL

/category/:catname

#Body

none

#Description

Get a list of all the toys in toys category, it will present 10 results per page.

#Method

GET

URL

/single/:id

#Body

none

#Description

Get a toy by id, will present 10 results per page.

#Method

POST

URL

/toys

#Body

{
"name": "cat",
"info": "cat",
"category": "Fun Category",
"img_url": "https://example.com/toy-image.jpg", /* not require */
"price": 45.7 ,
"created_date": "2023-12-03"/* not require */
}

#Description

Add a new toy to the data base, You need a token for this query

#Method

PATCH

URL

/toys/:editId

#Body

{
"name": "cat", /* not require */
"info": "cat game", /* not require */
"category": "Fun Category", /* not require */
"img_url": "https://example.com/toy-image.jpg",/* not require */
"price": 45.7 , /* not require */
"created_date": "2023-12-03" /* not require */
}

#Description

Update a toy in the data base, You need a token for this query.

#Method

DELETE

URL

/toys/delete/:delId

#Body

none

#Description

Delete a toy by id from the data base, You need a token for this query.
